LFPy_util package
=================

Utility package for LFPy. 

The modules add functions to aid data collection and plotting with LFPy 
and neuron. Functions can act as a guide for new code. 

Dependencides:
--------------

* matplotlib
* sklearn
* numpy
* neuron
* LFPy
* Latex -- For formatting text in plots. Texlive-full is sufficient.
* JSAnimation(optional) -- Animations in plot module. Is only imported when the
  animation is made. Can be found in the ext folder.
